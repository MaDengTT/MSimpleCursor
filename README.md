#Integration

Add jitpack.io repository to your root build.gradle:
```java
allprojects {
 repositories {
    jcenter()
    maven { url "https://jitpack.io" }
 }
}
```
Add the dependency to your module build.gradle:
```java
	dependencies {
	        compile 'com.github.MaDengTT:MSimpleCursor:0.2'
	}
```
